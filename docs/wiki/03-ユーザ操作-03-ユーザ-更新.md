# ユーザの更新 - users/{userId}/update

## 概要

APIを使用して既存のユーザ情報を更新します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<UpdateUserResponse>> UpdateUserAsync(
    long userId,
    string? name = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<UpdateUserResponse>> UpdateUserAsync(
    long userId,
    UpdateUserRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数      | 型        | 必須 | 説明                   |
|-----------|-----------|:----:|------------------------|
| `userId`  | long      | Yes  | ユーザID               |
| `name`    | string?   |      | 名前                   |
| `timeout` | TimeSpan? |      | リクエストタイムアウト |

## リクエストモデルのプロパティ

`UpdateUserRequest` は `CreateUserRequest` と同様のプロパティに加え、以下が含まれます：

| プロパティ | 型    | 説明         |
|------------|-------|--------------|
| `Lockout`  | bool? | ロックアウト |

## 使用例

### シンプルな更新

```csharp
var result = await client.UpdateUserAsync(
    userId: 123,
    name: "更新後の名前");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("ユーザ情報を更新しました");
}
```

### リクエストモデルを使用した更新

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Users;

var request = new UpdateUserRequest
{
    Name = "更新後の名前",
    DeptId = 2,
    Disabled = false,
    Lockout = false
};

var result = await client.UpdateUserAsync(123, request);
```

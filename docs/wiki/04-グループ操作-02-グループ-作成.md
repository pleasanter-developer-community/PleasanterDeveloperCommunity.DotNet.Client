# グループの作成 - groups/create

## 概要

APIを使用して新しいグループを作成します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<CreateGroupResponse>> CreateGroupAsync(
    string groupName,
    string? body = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<CreateGroupResponse>> CreateGroupAsync(
    CreateGroupRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `groupName` | string | Yes | グループ名 |
| `body` | string? | | 内容 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## リクエストモデルのプロパティ

`CreateGroupRequest` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `GroupName` | string? | グループ名 |
| `Body` | string? | 内容 |
| `Disabled` | bool? | 無効 |
| `GroupMembers` | List<GroupMemberRequest>? | グループメンバー |

## 使用例

### シンプルなグループ作成

```csharp
var result = await client.CreateGroupAsync(
    groupName: "開発チーム",
    body: "開発メンバーのグループ");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine($"作成されたグループID: {result.Data.Id}");
}
```

### メンバーを含むグループ作成

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Groups;

var request = new CreateGroupRequest
{
    GroupName = "プロジェクトチーム",
    Body = "プロジェクトメンバー",
    GroupMembers = new List<GroupMemberRequest>
    {
        new GroupMemberRequest { UserId = 1, Admin = true },
        new GroupMemberRequest { UserId = 2, Admin = false },
        new GroupMemberRequest { DeptId = 1 }  // 組織全体を追加
    }
};

var result = await client.CreateGroupAsync(request);
```

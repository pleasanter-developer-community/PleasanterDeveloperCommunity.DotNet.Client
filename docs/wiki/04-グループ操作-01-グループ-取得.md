# グループの取得 - groups/get

## 概要

APIを使用してグループ一覧を取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<GetGroupsResponse>> GetGroupsAsync(
    int? offset = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<GetGroupsResponse>> GetGroupsAsync(
    GetGroupsRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `offset` | int? | | 取得開始位置 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## レスポンス

`GroupData` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `GroupId` | int | グループID |
| `GroupName` | string? | グループ名 |
| `Body` | string? | 内容 |
| `Disabled` | bool | 無効フラグ |
| `GroupMembers` | List<GroupMemberData>? | グループメンバー |

## 使用例

### 全グループを取得

```csharp
var result = await client.GetGroupsAsync();

if (result.Data?.StatusCode == 200)
{
    foreach (var group in result.Data.Response?.Data ?? new List<GroupData>())
    {
        Console.WriteLine($"グループ: {group.GroupName}");
    }
}
```

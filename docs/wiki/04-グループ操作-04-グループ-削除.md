# グループの削除 - groups/{groupId}/delete

## 概要

APIを使用してグループを削除します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<DeleteGroupResponse>> DeleteGroupAsync(
    long groupId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<DeleteGroupResponse>> DeleteGroupAsync(
    long groupId,
    DeleteGroupRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `groupId` | long | ✓ | グループID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな削除

```csharp
var result = await client.DeleteGroupAsync(groupId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("グループを削除しました");
}
```

# 集計の同期 - sites/{siteId}/synchronizesummaries

## 概要

APIを使用してサイトの集計を同期します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サイトの管理権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<SynchronizeSummariesResponse>> SynchronizeSummariesAsync(
    long siteId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<SynchronizeSummariesResponse>> SynchronizeSummariesAsync(
    long siteId,
    SynchronizeSummariesRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `siteId` | long | ✓ | サイトID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 集計を同期

```csharp
var result = await client.SynchronizeSummariesAsync(siteId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("集計を同期しました");
}
```

# 検索インデックスの再構築 - sites/{siteId}/rebuildsearchindexes

## 概要

APIを使用してサイトの検索インデックスを再構築します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サイトの管理権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<RebuildSearchIndexesResponse>> RebuildSearchIndexesAsync(
    long siteId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<RebuildSearchIndexesResponse>> RebuildSearchIndexesAsync(
    long siteId,
    RebuildSearchIndexesRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `siteId` | long | ✓ | サイトID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 検索インデックスを再構築

```csharp
var result = await client.RebuildSearchIndexesAsync(siteId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("検索インデックスを再構築しました");
}
```

# 組織の取得 - depts/get

## 概要

APIを使用して組織一覧を取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<GetDeptsResponse>> GetDeptsAsync(
    int? offset = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<GetDeptsResponse>> GetDeptsAsync(
    GetDeptsRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `offset` | int? | | 取得開始位置 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## レスポンス

`DeptData` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `DeptId` | int | 組織ID |
| `DeptCode` | string? | 組織コード |
| `DeptName` | string? | 組織名 |
| `Body` | string? | 内容 |
| `Disabled` | bool | 無効フラグ |

## 使用例

### 全組織を取得

```csharp
var result = await client.GetDeptsAsync();

if (result.Data?.StatusCode == 200)
{
    foreach (var dept in result.Data.Response?.Data ?? new List<DeptData>())
    {
        Console.WriteLine($"組織: {dept.DeptName} ({dept.DeptCode})");
    }
}
```

# 組織の更新 - depts/{deptId}/update

## 概要

APIを使用して既存の組織情報を更新します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<UpdateDeptResponse>> UpdateDeptAsync(
    long deptId,
    string? deptName = null,
    string? body = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<UpdateDeptResponse>> UpdateDeptAsync(
    long deptId,
    UpdateDeptRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `deptId` | long | ✓ | 組織ID |
| `deptName` | string? | | 組織名 |
| `body` | string? | | 内容 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな更新

```csharp
var result = await client.UpdateDeptAsync(
    deptId: 123,
    deptName: "新しい組織名");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("組織情報を更新しました");
}
```

### リクエストモデルを使用した更新

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Depts;

var request = new UpdateDeptRequest
{
    DeptCode = "DEV002",
    DeptName = "開発第2部",
    Body = "更新後の説明"
};

var result = await client.UpdateDeptAsync(123, request);
```

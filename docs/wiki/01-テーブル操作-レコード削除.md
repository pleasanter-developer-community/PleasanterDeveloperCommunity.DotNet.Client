# レコード削除 - items/{recordId}/delete

## 概要

指定したレコードIDのレコードを削除します。

## メソッド

```csharp
Task<ApiResponse<DeleteRecordResponse>> DeleteRecordAsync(long recordId, TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `recordId` | long | ✓ | レコードID（IssueIdまたはResultId） |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 基本的な削除

```csharp
var result = await client.DeleteRecordAsync(recordId: 12345);

if (result.Success)
{
    Console.WriteLine($"削除されたレコードID: {result.Data.Id}");
    Console.WriteLine($"メッセージ: {result.Message}");
}
```

### タイムアウトを指定した削除

```csharp
var result = await client.DeleteRecordAsync(
    recordId: 12345,
    timeout: TimeSpan.FromSeconds(30)
);

if (result.Success)
{
    Console.WriteLine($"削除されたレコードID: {result.Data.Id}");
}
else
{
    Console.WriteLine($"エラー: {result.Message}");
}
```

## レスポンス

### DeleteRecordResponse

| プロパティ | 型 | 説明 |
|-----------|------|------|
| `Id` | long | 削除されたレコードID |
| `LimitPerDate` | int? | 1日あたりのAPI呼び出し上限 |
| `LimitRemaining` | int? | 残りのAPI呼び出し回数 |

### レスポンス例

```json
{
    "Id": 12345,
    "StatusCode": 200,
    "LimitPerDate": 10000,
    "LimitRemaining": 9995,
    "Message": "\" レコードのタイトル \" を削除しました。"
}
```

## 注意事項

- 削除したレコードは復元できません。削除前に十分確認してください。
- 適切な権限がない場合、削除操作は失敗します。
- 他のレコードからリンクされている場合、関連する参照も解除されます。

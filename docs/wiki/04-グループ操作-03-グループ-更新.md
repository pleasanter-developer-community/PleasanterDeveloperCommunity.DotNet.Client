# グループの更新 - groups/{groupId}/update

## 概要

APIを使用して既存のグループ情報を更新します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<UpdateGroupResponse>> UpdateGroupAsync(
    long groupId,
    string? groupName = null,
    string? body = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<UpdateGroupResponse>> UpdateGroupAsync(
    long groupId,
    UpdateGroupRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `groupId` | long | ✓ | グループID |
| `groupName` | string? | | グループ名 |
| `body` | string? | | 内容 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな更新

```csharp
var result = await client.UpdateGroupAsync(
    groupId: 123,
    groupName: "新しいグループ名");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("グループ情報を更新しました");
}
```

### リクエストモデルを使用した更新

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Groups;

var request = new UpdateGroupRequest
{
    GroupName = "更新後のグループ名",
    Body = "更新後の説明",
    GroupMembers = new List<GroupMemberRequest>
    {
        new GroupMemberRequest { UserId = 1, Admin = true }
    }
};

var result = await client.UpdateGroupAsync(123, request);
```

# 拡張機能の取得 - extensions/get

## 概要

APIを使用して拡張機能一覧を取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サービス管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<GetExtensionsResponse>> GetExtensionsAsync(
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<GetExtensionsResponse>> GetExtensionsAsync(
    GetExtensionsRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数      | 型        | 必須 | 説明                   |
|-----------|-----------|:----:|------------------------|
| `timeout` | TimeSpan? |      | リクエストタイムアウト |

## レスポンス

`ExtensionData` には以下のプロパティが含まれます：

| プロパティ             | 型      | 説明       |
|------------------------|---------|------------|
| `ExtensionId`          | long    | 拡張機能ID |
| `ExtensionType`        | string? | 種類       |
| `ExtensionName`        | string? | 拡張機能名 |
| `ExtensionDescription` | string? | 説明       |
| `ExtensionSettings`    | string? | 設定       |
| `Disabled`             | bool    | 無効フラグ |

## 使用例

### 拡張機能一覧を取得

```csharp
var result = await client.GetExtensionsAsync();

if (result.Data?.StatusCode == 200)
{
    foreach (var ext in result.Data.Response?.Data ?? new List<ExtensionData>())
    {
        Console.WriteLine($"拡張機能: {ext.ExtensionName} ({ext.ExtensionType})");
    }
}
```

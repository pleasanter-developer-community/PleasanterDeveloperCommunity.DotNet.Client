# 拡張機能の削除 - extensions/{extensionId}/delete

## 概要

APIを使用して拡張機能を削除します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サービス管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<DeleteExtensionResponse>> DeleteExtensionAsync(
    long extensionId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<DeleteExtensionResponse>> DeleteExtensionAsync(
    long extensionId,
    DeleteExtensionRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `extensionId` | long | Yes | 拡張機能ID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 拡張機能を削除

```csharp
var result = await client.DeleteExtensionAsync(extensionId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("拡張機能を削除しました");
}
```

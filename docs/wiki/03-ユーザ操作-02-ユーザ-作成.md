# ユーザの作成 - users/create

## 概要

APIを使用して新しいユーザを作成します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<CreateUserResponse>> CreateUserAsync(
    string loginId,
    string name,
    string password,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<CreateUserResponse>> CreateUserAsync(
    CreateUserRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `loginId` | string | Yes | ログインID |
| `name` | string | Yes | 名前 |
| `password` | string | Yes | パスワード |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## リクエストモデルのプロパティ

`CreateUserRequest` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `LoginId` | string? | ログインID |
| `Name` | string? | 名前 |
| `Password` | string? | パスワード |
| `UserCode` | string? | ユーザコード |
| `Birthday` | DateTime? | 生年月日 |
| `Gender` | string? | 性別 |
| `Language` | string? | 言語 |
| `TimeZone` | string? | タイムゾーン |
| `DeptId` | int? | 組織ID |
| `Body` | string? | 内容 |
| `TenantManager` | bool? | テナント管理者 |
| `ServiceManager` | bool? | サービス管理者 |
| `AllowApi` | bool? | APIを許可 |
| `Disabled` | bool? | 無効 |
| `MailAddresses` | List<string>? | メールアドレス |

## 使用例

### シンプルなユーザ作成

```csharp
var result = await client.CreateUserAsync(
    loginId: "user001",
    name: "テストユーザ",
    password: "Password123!");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine($"作成されたユーザID: {result.Data.Id}");
}
```

### リクエストモデルを使用した詳細な作成

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Users;

var request = new CreateUserRequest
{
    LoginId = "user002",
    Name = "詳細ユーザ",
    Password = "Password123!",
    UserCode = "U002",
    DeptId = 1,
    TenantManager = false,
    AllowApi = true,
    MailAddresses = new List<string> { "user002@example.com" }
};

var result = await client.CreateUserAsync(request);
```

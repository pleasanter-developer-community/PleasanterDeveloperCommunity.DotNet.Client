# 組織の作成 - depts/create

## 概要

APIを使用して新しい組織を作成します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<CreateDeptResponse>> CreateDeptAsync(
    string deptCode,
    string deptName,
    string? body = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<CreateDeptResponse>> CreateDeptAsync(
    CreateDeptRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `deptCode` | string | ✓ | 組織コード |
| `deptName` | string | ✓ | 組織名 |
| `body` | string? | | 内容 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな組織作成

```csharp
var result = await client.CreateDeptAsync(
    deptCode: "DEV001",
    deptName: "開発部",
    body: "ソフトウェア開発部門");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine($"作成された組織ID: {result.Data.Id}");
}
```

### リクエストモデルを使用した作成

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Depts;

var request = new CreateDeptRequest
{
    DeptCode = "SALES001",
    DeptName = "営業部",
    Body = "営業部門",
    Disabled = false
};

var result = await client.CreateDeptAsync(request);
```

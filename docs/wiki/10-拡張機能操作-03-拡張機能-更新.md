# 拡張機能の更新 - extensions/{extensionId}/update

## 概要

APIを使用して既存の拡張機能を更新します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サービス管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<UpdateExtensionResponse>> UpdateExtensionAsync(
    long extensionId,
    string? extensionName = null,
    string? extensionSettings = null,
    string? extensionDescription = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<UpdateExtensionResponse>> UpdateExtensionAsync(
    long extensionId,
    UpdateExtensionRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `extensionId` | long | ✓ | 拡張機能ID |
| `extensionName` | string? | | 拡張機能名 |
| `extensionSettings` | string? | | 設定 |
| `extensionDescription` | string? | | 説明 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 拡張機能を更新

```csharp
var result = await client.UpdateExtensionAsync(
    extensionId: 123,
    extensionName: "更新後の拡張機能名",
    extensionSettings: "SELECT Id, Title FROM Items");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("拡張機能を更新しました");
}
```

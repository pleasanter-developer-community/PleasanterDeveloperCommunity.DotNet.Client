# バイナリストリームの取得 - binaries/{guid}/getstream

## 概要

APIを使用してバイナリデータをストリームとして取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<GetBinaryStreamResponse>> GetBinaryStreamAsync(
    string guid,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<GetBinaryStreamResponse>> GetBinaryStreamAsync(
    string guid,
    GetBinaryStreamRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `guid` | string | Yes | バイナリのGUID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## レスポンス

`BinaryStreamResponseData` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `Base64` | string? | Base64エンコードされたバイナリデータ |
| `ContentType` | string? | コンテンツタイプ |
| `FileName` | string? | ファイル名 |

## 使用例

### バイナリストリームを取得

```csharp
var result = await client.GetBinaryStreamAsync(guid: "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx");

if (result.Data?.StatusCode == 200 && result.Data.Response?.Base64 != null)
{
    var bytes = Convert.FromBase64String(result.Data.Response.Base64);
    await File.WriteAllBytesAsync(result.Data.Response.FileName ?? "downloaded_file", bytes);
    Console.WriteLine("ファイルをダウンロードしました");
}
```

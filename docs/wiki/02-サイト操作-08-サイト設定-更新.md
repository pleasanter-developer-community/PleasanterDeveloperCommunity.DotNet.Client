# サイト設定の更新 - sites/{siteId}/updatesitesettings

## 概要

APIを使用してサイト設定を更新します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サイトの管理権限が必要です

## メソッド

```csharp
// リクエストモデル版
Task<ApiResponse<UpdateSiteSettingsResponse>> UpdateSiteSettingsAsync(
    long siteId,
    UpdateSiteSettingsRequest request,
    TimeSpan? timeout = null)

// パラメータ指定版
Task<ApiResponse<UpdateSiteSettingsResponse>> UpdateSiteSettingsAsync(
    long siteId,
    object siteSettings,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `siteId` | long | ✓ | サイトID |
| `siteSettings` | object | ✓ | サイト設定オブジェクト |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### サイト設定を更新

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Sites;

var request = new UpdateSiteSettingsRequest
{
    SiteSettings = new
    {
        GridPageSize = 50,
        EditorColumns = new[] { "Title", "Body", "Status" }
    }
};

var result = await client.UpdateSiteSettingsAsync(siteId: 123, request);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("サイト設定を更新しました");
}
```

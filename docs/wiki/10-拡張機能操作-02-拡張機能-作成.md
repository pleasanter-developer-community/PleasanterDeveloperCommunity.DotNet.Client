# 拡張機能の作成 - extensions/create

## 概要

APIを使用して新しい拡張機能を作成します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サービス管理者権限が必要です

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<CreateExtensionResponse>> CreateExtensionAsync(
    string extensionType,
    string extensionName,
    string extensionSettings,
    string? extensionDescription = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<CreateExtensionResponse>> CreateExtensionAsync(
    CreateExtensionRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `extensionType` | string | Yes | 種類 |
| `extensionName` | string | Yes | 拡張機能名 |
| `extensionSettings` | string | Yes | 設定 |
| `extensionDescription` | string? | | 説明 |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### 拡張機能を作成

```csharp
var result = await client.CreateExtensionAsync(
    extensionType: "Sql",
    extensionName: "カスタムSQL",
    extensionSettings: "SELECT * FROM Items",
    extensionDescription: "テスト用SQL拡張機能");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine($"作成された拡張機能ID: {result.Data.Id}");
}
```

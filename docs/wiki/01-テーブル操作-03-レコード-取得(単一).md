# 単一レコード取得 - items/{recordId}/get

## 概要

指定したレコードIDのレコードを1件取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<RecordResponse>> GetRecordAsync(long recordId, View? view = null, TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<RecordResponse>> GetRecordAsync(long recordId, GetRecordRequest request, TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `recordId` | long | ✓ | レコードID（IssueIdまたはResultId） |
| `view` | View | | ビュー設定（取得する列の指定など） |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### パラメータ指定版

```csharp
// 単一レコードの取得
var record = await client.GetRecordAsync(recordId: 123);
```

### リクエストモデル版

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Items;

// リクエストモデルを使用した取得
var request = new GetRecordRequest
{
    View = new View
    {
        GridColumns = new List<string> { "Title", "Body", "ClassA" }
    }
};

var record = await client.GetRecordAsync(recordId: 123, request: request);
```

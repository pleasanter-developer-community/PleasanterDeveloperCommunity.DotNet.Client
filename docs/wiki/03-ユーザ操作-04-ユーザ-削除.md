# ユーザの削除 - users/{userId}/delete

## 概要

APIを使用してユーザを削除します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です
- 自分自身は削除できません

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<DeleteUserResponse>> DeleteUserAsync(
    long userId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<DeleteUserResponse>> DeleteUserAsync(
    long userId,
    DeleteUserRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `userId` | long | ✓ | ユーザID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな削除

```csharp
var result = await client.DeleteUserAsync(userId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("ユーザを削除しました");
}
```

### リクエストモデルを使用した削除

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Users;

var request = new DeleteUserRequest();
var result = await client.DeleteUserAsync(123, request);
```

# ライセンス情報の取得 - utility/getlicenseinfo

## 概要

APIを使用してプリザンターのライセンス情報を取得します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- サービス管理者権限が必要な場合があります

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<GetLicenseInfoResponse>> GetLicenseInfoAsync(
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<GetLicenseInfoResponse>> GetLicenseInfoAsync(
    GetLicenseInfoRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## レスポンス

`LicenseInfoData` には以下のプロパティが含まれます：

| プロパティ | 型 | 説明 |
|------------|------|------|
| `LicenseName` | string? | ライセンス名 |
| `Licensee` | string? | ライセンシー |
| `Expiration` | DateTime? | 有効期限 |
| `UserLimit` | int? | ユーザ数上限 |
| `SiteLimit` | int? | サイト数上限 |

## 使用例

### ライセンス情報を取得

```csharp
var result = await client.GetLicenseInfoAsync();

if (result.Data?.StatusCode == 200 && result.Data.Response?.Data != null)
{
    var info = result.Data.Response.Data;
    Console.WriteLine($"ライセンス名: {info.LicenseName}");
    Console.WriteLine($"ライセンシー: {info.Licensee}");
    Console.WriteLine($"有効期限: {info.Expiration}");
    Console.WriteLine($"ユーザ数上限: {info.UserLimit}");
    Console.WriteLine($"サイト数上限: {info.SiteLimit}");
}
```

# 組織の削除 - depts/{deptId}/delete

## 概要

APIを使用して組織を削除します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## 制限事項

- テナント管理者権限が必要です
- 所属するユーザがいる場合は削除できません

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<DeleteDeptResponse>> DeleteDeptAsync(
    long deptId,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<DeleteDeptResponse>> DeleteDeptAsync(
    long deptId,
    DeleteDeptRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数 | 型 | 必須 | 説明 |
|------|------|:----:|------|
| `deptId` | long | Yes | 組織ID |
| `timeout` | TimeSpan? | | リクエストタイムアウト |

## 使用例

### シンプルな削除

```csharp
var result = await client.DeleteDeptAsync(deptId: 123);

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("組織を削除しました");
}
```

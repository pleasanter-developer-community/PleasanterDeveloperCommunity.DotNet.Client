# メールの送信 - items/{itemId}/mails/send

## 概要

APIを使用してアイテムに関連するメールを送信します。

## 対応バージョン

- プリザンター 1.3.13.0以降
  - このライブラリの制約によるもの

## メソッド

```csharp
// パラメータ指定版
Task<ApiResponse<SendMailResponse>> SendMailAsync(
    long itemId,
    string to,
    string title,
    string body,
    string? cc = null,
    string? bcc = null,
    TimeSpan? timeout = null)

// リクエストモデル版
Task<ApiResponse<SendMailResponse>> SendMailAsync(
    long itemId,
    SendMailRequest request,
    TimeSpan? timeout = null)
```

## パラメータ

| 引数      | 型        | 必須 | 説明                   |
|-----------|-----------|:----:|------------------------|
| `itemId`  | long      | Yes  | アイテムID             |
| `to`      | string    | Yes  | 宛先                   |
| `title`   | string    | Yes  | 件名                   |
| `body`    | string    | Yes  | 本文                   |
| `cc`      | string?   |      | CC                     |
| `bcc`     | string?   |      | BCC                    |
| `timeout` | TimeSpan? |      | リクエストタイムアウト |

## 使用例

### メールを送信

```csharp
var result = await client.SendMailAsync(
    itemId: 123,
    to: "recipient@example.com",
    title: "テストメール",
    body: "これはテストメールです。");

if (result.Data?.StatusCode == 200)
{
    Console.WriteLine("メールを送信しました");
}
```

### CCとBCCを含めたメール送信

```csharp
using PleasanterDeveloperCommunity.DotNet.Client.Models.Requests.Mails;

var request = new SendMailRequest
{
    To = "recipient@example.com",
    Cc = "cc@example.com",
    Bcc = "bcc@example.com",
    Title = "重要なお知らせ",
    Body = "詳細な内容をこちらに記載します。"
};

var result = await client.SendMailAsync(123, request);
```
